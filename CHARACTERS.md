## Character Attributes

Each character in the database has the following attributes used for the guessing game:

| Attribute | Description |
|-----------|-------------|
| `is_male` | Character is male |
| `is_human` | Character is human |
| `is_alien_or_creature` | Character is an alien or creature |
| `is_cyborg_or_robot` | Character is a cyborg or robot |
| `has_black_hair` | Character has black hair |
| `has_blonde_hair` | Character has blonde hair |
| `has_blue_hair` | Character has blue hair |
| `has_red_hair` | Character has red hair |
| `wears_glasses` | Character wears glasses |
| `has_scar` | Character has visible scars |
| `wears_hat_or_mask` | Character wears a hat or mask |
| `uses_swords` | Character uses swords |
| `uses_guns` | Character uses guns |
| `has_superpowers` | Character has supernatural abilities |
| `can_fly` | Character can fly |
| `elemental_powers` | Character has elemental powers |
| `is_student` | Character is a student |
| `is_captain_or_leader` | Character is a captain or leader |
| `is_villain_or_antagonist` | Character is a villain |
| `is_intelligent` | Character is notably intelligent |
| `is_orphan` | Character is an orphan |
| `is_cold_or_stoic` | Character has cold/stoic personality |
| `is_pirate` | Character is a pirate |
| `is_ninja` | Character is a ninja |
| `is_detective_or_police` | Character is detective/police |
| `from_popular_shonen` | Character is from popular shonen anime |
| `is_isekai` | Character is from isekai genre |
